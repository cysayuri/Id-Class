Coloração Muda a cor do texto de um elemento.
Olá, mundo!
💻
clique para revelar o código
📄 Código
/* HTML */
          <p class="meu-texto">
            Olá, mundo!
          </p>

          /* CSS */
          .meu-texto {
            color: #1D9E75;
          }
resultado → texto em verde
